## Leveling the Frets

The goal is to get all the frets and the exact same height.  Mostly they are, but there some variations due to different wood density along the fretboard and varying installation pressure.

This is crucial to avoid string buzz, misfretting and other playability issues.

With the truss rod adjusted so the fretboard is perfectly flat, a file is run over them to remove any high spots.  Each fret has a line of Sharpie marker across it to identify which frets are getting hit.  Once all the marker lines are rubbed off they are all the same height.

{{<imgresize leveling.jpg "800x800" "Leveling the frets">}}

Fortunately there is not much work needed since I was careful during fret installation.

Then each fret is checked with a rocker (card scraper) to check its height relative to its two neighbors.  If it rocks, it is high.

{{<imgresize rocker.jpg "800x800" "Check for high spots">}}

Any high spots are fine tuned with a small file.

{{<imgresize fine_file.jpg "800x800" "Fine tuning the fret heights with a small file">}}

## Crowning

Now the frets are all the same height, but flat on top.  Time to recreate the rounded top (crown) across them.

This is done with a concave-shaped crowning file (medium width burr).

{{<imgresize crowning.jpg "800x800" "Crowning the frets">}}

{{<imgresize crowning_done.jpg "800x800" "Frets leveled and crowned">}}

## Polish

Once the crowns are reestablished it is time to sand out all the file marks and polish up the frets.  I start with 320 grit.  Then down to 600, 1,000, and all the way to 12,000.  1,500 through 12,000 are done with micro mesh polishing pads.

{{<imgresize micro_mesh.jpg "800x800" "Micro mesh polishing pads">}}

Finally I go over them with a buffing wheel on the dremel.

{{<imgresize polished.jpg "800x800" "Polished frets">}}

This process gets them to a mirror like finish.  The photo does not do them justice, but they are really slick and should play nicely.

The black residue from the metal polishing will come right off with some naptha.

## Stainless Steel Frets

This is my first time using stainless steel frets.  I was a bit worried since they are much harder than standard nickel-silver and allegedly much more difficult to work and rough on the tools.  I did not really find this to be the case.  They are harder and take a few more file strokes and sanding passes to remove material, but it is not a big deal at all.

Supposedly they do not wear out.

I like them.
