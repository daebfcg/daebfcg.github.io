Circuit Test Box

New and improved circuit experimentation kit

2026-5-10 12:54

New circuit testing rig made from a cigar box.

The inside is fully shielded.

Some mini breadboards for making and connecting modules. The rear-left one is intended to be the main output, and is connected to the output jack in the back. It also has the kill-switch wired to it which comes in handy while arranging connections.

I installed a little handle to lift up the inner lid. I removed it later realizing I can just use one of the attached knobs.

The output jack:

The components are attached from the inside and their wires go up through the holes to be connected to breadboards. I could use some more holes. I made some holes 1/4 inch for the mini switches and some 3/8 for the pots and rotary switches. I probably should have made them all 3/8. Also could use some 1/2 inch holes. Always easier to add more holes and make existing ones bigger.

This also serves as a storage compartment.

It is meant to be able to be closed up, for storage or even more shielding. One design flaw here is that it cannot be fully closed with wires and components sticking up without mashing them down.

My circuit

This is much neater and easier to deal with than the previous modular system. Although, this limits the number of modules to about five or six.

Dial Plates

I like how these dial plates turned out. I am using the set-screw chicken-head knobs for pots that have solid non-splined shafts, and press-on knobs for pots that have the splined shafts. I do like to know what number I am on. Unfortunately set-screws knobs for solid shafts that also have numbers are rare.

The plates for the rotary switches have 12 positions. The plates for the pots go from 0 to 10.

In the image above, the white chicken-head is for selecting from three different capacitors for the main (treble) tone control, and the beige knob adjusts the filtering amount.
